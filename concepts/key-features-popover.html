1 · Fade in cell

Truncated cells show a soft vertical fade at the bottom so the eye reads "there's more". On hover, a tiny "+N more" pill appears.

2 · Click opens popover

Clicking anywhere in a features cell opens an anchored card with the full bullet list. Morpho-Blue header strip echoes the design-system card pattern.

3 · Edit in the popover

The popover is where editing happens — roomy, readable, autosaves on close. Esc / outside click / × button all dismiss.
